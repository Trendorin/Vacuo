# Support

- Bugs and reproducible compatibility problems: [issue tracker](https://github.com/Trendorin/Vacuo/issues/new/choose)
- Security vulnerabilities: [private security advisory](https://github.com/Trendorin/Vacuo/security/advisories/new)
- Usage questions: [GitHub Discussions](https://github.com/Trendorin/Vacuo/discussions)

Before reporting a cleanup result, run `vacuoctl system` and `vacuoctl scan --json`. Remove usernames, home paths and unrelated personal data from the output.

Vacuo is a community open-source project without a guaranteed support SLA.
